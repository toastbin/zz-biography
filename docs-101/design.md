# 项目设计文档

## 项目概述

**zz-biography** 是一个角色传记 Web 应用，基于 Vue 3 + TypeScript 构建。用户可以选择角色，查看角色背景故事。

## 目标

- 提供沉浸式的角色选择体验
- 支持多角色扩展，数据与视图解耦
- 保持轻量，无后端依赖，纯前端运行

---

## 页面结构

```
/select          角色选择页（当前实现）
/biography/:id   角色传记详情页（待开发）
```

### 角色选择页（/select）

```
┌─────────────────────────────────┐
│         选择你的角色             │
│       开启属于你的冒险之旅        │
├─────────────────────────────────┤
│  ┌───────┐ ┌───────┐ ┌───────┐  │
│  │  ⚔️   │ │  🔮   │ │  🏹   │  │
│  │ 战士  │ │ 法师  │ │ 游侠  │  │
│  │ ...  │ │ ...  │ │ ...  │  │
│  └───────┘ └───────┘ └───────┘  │
├─────────────────────────────────┤
│           [ 确认选择 ]           │
└─────────────────────────────────┘
```

**交互逻辑：**
1. 默认无选中，确认按钮置灰
2. 点击卡片 → 高亮选中，记录 `selectedCharacter`
3. 再次点击已选中卡片 → 保持选中（不取消）
4. 点击确认 → 跳转 `/biography/:id`（待实现）

## 数据模型

```typescript
interface Character {
  id: string          // 唯一标识，用于路由参数
  name: string        // 显示名称
  avatar: string      // emoji 或图片路径
  description: string // 简介文字
}
```

**当前角色数据**（`src/data/characters.ts`）：

| id | name | avatar |
|----|------|--------|
| warrior | 战士 | ⚔️ |
| mage | 法师 | 🔮 |
| ranger | 游侠 | 🏹 |

## 组件设计

### CharacterSelect.vue（页面）

- 职责：数据编排、状态管理、页面布局
- 状态：`selectedCharacter: Ref<Character | null>`
- 不直接处理样式细节，委托给子组件

### CharacterCard.vue（组件）

- 职责：单张卡片的展示与交互
- 纯展示组件，无内部状态
- Props：`character`、`selected`
- Emits：`select(character)`
